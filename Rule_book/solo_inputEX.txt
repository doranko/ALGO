~ ALGO 1 人プレイの入力に関して ~

カードを置く時の入力は「a 段目の左から b 番目」

e.g.
+-------------+
|  B3  B4     |
|  B2  W3     |
|  B1         |
|  W8  W9 W10 |
|  W1         |
| W11         |
+-------------+
  の時, 入力が「3 2」カードが「W2」ならば, 結果は
+-------------+
|  B3  B4     |
|  B2  W3     |
|  B1  W2     |
|  W8  W9 W10 |
|  W1         |
| W11         |
+-------------+
  となる.
